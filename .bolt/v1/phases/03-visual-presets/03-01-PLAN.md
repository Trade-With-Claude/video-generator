# Phase 3 Plan: Visual Presets

## Acceptance Criteria

### AC1: Three distinct presets render successfully
- **Given** the preset system
- **When** I render 5 seconds each with mood ambient, focus, and trippy
- **Then** three visually distinct MP4s are produced (different file sizes, confirming different content)

### AC2: Presets define complete visual identity
- **Given** a preset name
- **When** I load it
- **Then** it provides: color palette, noise params, particle config, motion speed — all fields populated

### AC3: Seed-based randomization produces variety
- **Given** the `ambient` preset with seed=1 and seed=2
- **When** I render both
- **Then** the outputs differ (different file hashes) while maintaining the same mood/palette family

## Tasks

### Task 1: Preset data model + three presets
- **files:** `src/video_generator/presets.py`, `pyproject.toml`
- **action:** Create a `Preset` dataclass holding all visual parameters: background colors (low/high), noise scale/speed, particle count/colors/sizes/drift/alpha, motion speed. Define three presets: `ambient` (deep blues/teals, slow, gentle drift, 1000 particles), `focus` (warm ambers/golds on dark bg, medium speed, structured flow, 800 particles), `trippy` (vibrant multi-color, fast motion, dense 3000 particles, wide drift). Add `get_preset(name, seed)` function that returns a configured Preset with seed-based randomization within palette bounds. Also add `scipy` to `pyproject.toml` (carry-forward from Phase 2).
- **verify:** `python3 -c "from video_generator.presets import get_preset; p = get_preset('ambient'); print(p)"`
- **done:** true

### Task 2: Preset-to-layers builder + render script
- **files:** `src/video_generator/render.py`, `scripts/test_presets.py`
- **action:** `build_layers(preset, width, height)` function that takes a Preset and returns a list of configured Layer instances (NoiseBackground + ParticleLayer). Test script renders 5 seconds of each mood to `output/preset_{mood}.mp4`, printing which preset is being rendered.
- **verify:** `python3 scripts/test_presets.py` produces three MP4s in `output/`
- **done:** true
