# Phase 1 Plan: Project Foundation

## Acceptance Criteria

### AC1: Package installs cleanly
- **Given** a fresh venv
- **When** I run `pip install -e .`
- **Then** all dependencies install and `video_generator` is importable

### AC2: Config loads with defaults
- **Given** no env vars or overrides
- **When** I instantiate `Settings()`
- **Then** I get 1920x1080, 30fps, `./output` dir, and all defaults populated

### AC3: FFmpeg pipeline produces a valid MP4
- **Given** FFmpeg is installed
- **When** I run the test script that pipes solid-color frames
- **Then** a valid 10-second MP4 is created at 1080p30

### AC4: FFmpeg check fails gracefully
- **Given** FFmpeg is not on PATH
- **When** the availability check runs
- **Then** a clear error message is raised (not a cryptic subprocess error)

## Tasks

### Task 1: Package structure + config
- **files:** `pyproject.toml`, `src/video_generator/__init__.py`, `src/video_generator/config.py`
- **action:** Create Python package with src layout. `pyproject.toml` with dependencies (pycairo, numpy, opensimplex, questionary, pydantic-settings). Config via Pydantic Settings: resolution (w/h), fps, output_dir, loop_duration, seed.
- **verify:** `pip install -e .` succeeds, `python -c "from video_generator.config import Settings; print(Settings())"`
- **done:** true

### Task 2: FFmpeg wrapper + check
- **files:** `src/video_generator/ffmpeg.py`
- **action:** `check_ffmpeg()` function using `shutil.which`. `FFmpegPipe` class that opens an FFmpeg subprocess with stdin pipe, accepts raw RGB frames as bytes/numpy arrays, closes cleanly. YouTube-optimized H.264 defaults (yuv420p, crf 18, faststart).
- **verify:** `python -c "from video_generator.ffmpeg import check_ffmpeg; check_ffmpeg()"`
- **done:** true

### Task 3: Test script — solid-color MP4
- **files:** `scripts/test_pipeline.py`
- **action:** Script that creates a solid-color numpy frame (1080p), pipes 300 frames (10s at 30fps) through `FFmpegPipe`, outputs `output/test.mp4`. Validates the file exists and has nonzero size.
- **verify:** `python scripts/test_pipeline.py` produces a playable `output/test.mp4`
- **done:** true

## Boundaries
- **DO NOT CHANGE:** `.bolt/` files (except STATE.md updates), `.git/`
- **OUT OF SCOPE:** Cairo rendering, noise fields, particles, presets, TUI — all later phases
