## Tasks

### Task 1: Time system + noise field + layer compositor
- **files:** `src/video_generator/time_loop.py`, `src/video_generator/layers.py`
- **action:** `TimeLoop` class: given fps + duration, yields `(frame_index, t, theta)` tuples where `t ‚ąą [0,1)` and `theta = t * 2ŌÄ`. `NoiseBackground` layer: generates opensimplex noise at 270p using `noise4array` with circular z/w coords (`cos(őł)`, `sin(őł)`), bicubic upscale to 1080p, maps to RGB gradient. `LayerCompositor`: takes a list of layers, calls each with `(t, theta, canvas)`, composites results into a final RGB frame.
- **verify:** `python3 -c "from video_generator.time_loop import TimeLoop; frames = list(TimeLoop(30, 2.0)); print(len(frames), frames[0], frames[-1])"`
- **done:** true

### Task 2: Particle system + Cairo renderer
- **files:** `src/video_generator/particles.py`
- **action:** `ParticleLayer` class: initializes N particles with random positions, phases, sizes, colors as NumPy arrays. On each frame, updates positions using periodic functions (`sin(őł + phase)`, `cos(őł + phase)`) for seamless looping. - **verify:** `python3 -c "from video_generator.particles import ParticleLayer; p = ParticleLayer(1920, 1080); print(p.render(0.5, 3.14).shape)"`
- **done:** true

### Task 3: Integration test ‚ÄĒ render a 30-second seamless loop
