# Phase 5 Plan: TUI & Polish

## Acceptance Criteria

### AC1: Interactive TUI produces a video end-to-end
- **Given** the CLI is run interactively
- **When** I select a mood, set duration, and confirm
- **Then** a video is generated with the chosen settings

### AC2: Python API is callable programmatically
- **Given** the `generate()` function
- **When** I call `generate(mood="ambient", target_duration=30, loop_duration=5)`
- **Then** it returns the path to the output MP4

### AC3: CLI entry point works via `python -m video_generator`
- **Given** the package is installed
- **When** I run `python -m video_generator --mood ambient --duration 30 --loop-duration 5`
- **Then** a video is generated without interactive prompts

## Tasks

### Task 1: `generate()` API + CLI entry point
- **files:** `src/video_generator/generate.py`, `src/video_generator/__main__.py`, `pyproject.toml`
- **action:** `generate()` function that orchestrates the full pipeline: get preset → build layers → render loop → extend to target → return output path. Prints progress (frame count, ETA). `__main__.py` parses args with argparse for non-interactive use (`--mood`, `--duration`, `--loop-duration`, `--seed`, `--output`). Add `[project.scripts]` entry in pyproject.toml for `video-generator` CLI command.
- **verify:** `python3 -c "from video_generator.generate import generate; p = generate(mood='ambient', target_duration=15, loop_duration=5); print(p)"`
- **done:** true

### Task 2: Interactive TUI with questionary
- **files:** `src/video_generator/tui.py`, `src/video_generator/__main__.py` (update)
- **action:** `run_tui()` function: select mood (ambient/focus/trippy), input target duration (default 300), input loop duration (default 45), optional seed, confirm & generate. Progress display during render. `__main__.py` launches TUI when no args given, CLI mode when args provided.
- **verify:** `python3 -m video_generator --mood focus --duration 15 --loop-duration 5` produces a video
- **done:** true
